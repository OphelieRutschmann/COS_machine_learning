# Applied Data Sciences: Machine Learning

#### Author
Ophélie Rutschmann

#### Description
This repository contains the project done during the COS in applied data sciences: machine learning from the EPFL extension school. 
This certificat consists of 15 ECTS and is divided into four modules, each with their own final project:
1) Introduction to data analysis with Python
2) Applied data analysis
3) Applied machine learning 1
4) Applied machine learning 2

The main coding language is Python, and the projects are presented as jupyter lab notebooks. 

At the end of the course, there is a capstone project that will also be part of this repository.
